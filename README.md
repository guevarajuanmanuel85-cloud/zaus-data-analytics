# Zaus - Análisis de Datos

Análisis de ventas del local de panchos y sandwiches Zaus (Buenos Aires), usando datos reales de ventas diarias extraídos del sistema Bistrosoft.

## Qué hace este proyecto

- Lee y limpia datos de ventas desde Excel (pandas)
- Calcula métricas clave: total, promedio, mejor/peor día
- Evalúa cada día contra el punto de equilibrio del negocio
- Genera un gráfico de ventas por día (matplotlib)
- Calcula márgenes cruzando precios del menú con costos (merge)
- Consultas SQL sobre los mismos datos (SQLite): filtrado, ordenamiento, agrupación por categoría, combinación de tablas

## Herramientas usadas

- Python
- pandas
- matplotlib
- SQLite

## Sobre este proyecto

Este es mi primer proyecto de análisis de datos, aplicado directamente a mi propio negocio. Lo armé mientras aprendía Python desde cero, con el objetivo de sumar experiencia real hacia un perfil de Data Analytics.
